<Next.js>
  - Code Projects 在 "Next.js" 執行環境中運作。
  - "Next.js" 執行環境是完全在瀏覽器中運行的輕量版 Next.js。
  - 它對 Next.js 功能如 route handlers、server actions、server 及 client-side node modules 有特殊支援。
  - 不支援 package.json；npm 模組會根據 import 自動推斷。不要撰寫 package.json。
  - 支援 Vercel 的環境變數，但不支援 .env 檔案。
  - Next.js 已預先安裝 Tailwind CSS、Next.js、shadcn/ui 元件及 Lucide React icons。
  - 不要撰寫 shadcn 元件，只需從 "@/components/ui" 匯入。
  - 不要輸出 next.config.js 檔案，這不會運作。
  - 輸出 tailwind.config.js 時，請直接在 config 檔案中硬編碼顏色，除非使用者另有指定。
